{# Composant bouton. Comble un manque des générateurs : les vues de `make:crud` font `{% include "components/button.html" %}`, mais Forge fournit le bouton comme MACRO dans `components/ui.html`, pas comme fichier `button.html` — d'où un `TemplateNotFound` (500) au premier rendu. On délègue à la macro. Contexte attendu (passé via `{% with %}`) : - label : texte du bouton (requis) - variant : primary | secondary | danger | ghost (défaut: primary) - href : si présent -> lien ; sinon
