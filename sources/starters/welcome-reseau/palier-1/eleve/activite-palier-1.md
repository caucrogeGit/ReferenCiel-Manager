# Activité : fabriquer et tester un câble Ethernet droit T568B

## Principe de l'activité

Cette activité se réalise à partir du dossier technique du palier 1.

Le dossier technique contient les informations nécessaires pour réussir le travail demandé.
L'activité ne redonne pas les explications techniques.

Vous devez donc consulter le dossier technique lorsque vous avez besoin d'une information.

## Travail demandé

Vous devez fabriquer un câble Ethernet droit en norme T568B.

Le câble devra ensuite être testé avec un testeur RJ45.

Le palier est terminé uniquement lorsque le câble est conforme.

## Matériel nécessaire

Préparez le poste de travail avec le matériel suivant :

- un morceau de câble Ethernet ;
- deux connecteurs RJ45 ;
- une pince à dénuder ;
- une pince à sertir RJ45 ;
- un testeur RJ45 ;
- le dossier technique du palier 1.

Si un élément manque, ne commencez pas l'activité.

## Demander de l'aide

Vous pouvez demander de l'aide, mais la demande doit être formulée correctement.

Avant d'appeler le professeur, vous devez pouvoir expliquer clairement :

- l'étape sur laquelle vous travaillez ;
- ce que vous avez déjà essayé ;
- ce que vous avez observé ;
- ce qui ne fonctionne pas ;
- la partie du dossier technique que vous avez consultée.

Une demande d'aide ne doit pas être formulée comme ceci :

```text
Je ne comprends rien.
Ça ne marche pas.
Je ne sais pas quoi faire.
```

Une demande d'aide doit être formulée comme ceci :

```text
Je suis à l'étape du sertissage du premier connecteur.
J'ai rangé les fils selon la norme demandée, mais je ne suis pas sûr du sens du connecteur.
J'ai consulté la partie sur la norme T568B et la partie sur le connecteur RJ45.
J'ai besoin d'aide pour vérifier l'ordre des fils avant de sertir.
```

Le professeur aide plus efficacement lorsque le problème est expliqué avec précision.

Un technicien ne dit pas seulement que ça ne fonctionne pas.
Il explique ce qu'il a fait, ce qu'il a observé et ce qu'il veut vérifier.

## Étape 1 : préparer le câble

Coupez le câble à la longueur demandée.

Dénudez la gaine extérieure.

Vérifiez l'état des fils internes avant de continuer.

En cas de doute, consultez le dossier technique.

## Étape 2 : préparer les fils

Séparez les paires.

Organisez les fils selon la norme demandée.

Avant d'insérer les fils dans le connecteur, vérifiez l'ordre avec le dossier technique.

Ne passez pas à l'étape suivante si l'ordre n'est pas correct.

## Étape 3 : placer le premier connecteur RJ45

Insérez les fils dans le connecteur RJ45.

Avant de sertir, vérifiez :

- l'ordre des fils ;
- l'insertion des fils ;
- la position de la gaine ;
- le sens du connecteur.

Lorsque tout est correct, sertissez le connecteur.

## Étape 4 : placer le deuxième connecteur RJ45

Réalisez la deuxième extrémité du câble.

Le câble demandé est un câble droit.

Avant de sertir, vérifiez que la deuxième extrémité respecte bien la consigne de l'activité.

Lorsque tout est correct, sertissez le connecteur.

## Étape 5 : tester le câble

Branchez les deux extrémités du câble sur le testeur RJ45.

Lancez le test.

Observez le résultat.

Comparez le résultat obtenu avec le résultat attendu dans le dossier technique.

## Étape 6 : corriger si nécessaire

Si le câble n'est pas conforme, ne recommencez pas au hasard.

Identifiez le type d'erreur à partir du testeur.

Consultez la partie du dossier technique sur les erreurs fréquentes.

Corrigez uniquement ce qui doit être corrigé.

Si une extrémité sertie est incorrecte, coupez cette extrémité et recommencez avec un nouveau connecteur RJ45.

Testez à nouveau le câble après correction.

## Résultat attendu

À la fin de l'activité :

- le câble est terminé ;
- les deux connecteurs sont sertis correctement ;
- le câble a été testé ;
- le résultat du testeur est conforme ;
- le câble peut être présenté au professeur pour validation du palier.

## Validation du palier

Le palier 1 est validé lorsque le câble fonctionne au testeur RJ45.

Si le câble n'est pas conforme, le palier n'est pas terminé.

Vous devez corriger le câble, refaire le test, puis demander une nouvelle vérification.
