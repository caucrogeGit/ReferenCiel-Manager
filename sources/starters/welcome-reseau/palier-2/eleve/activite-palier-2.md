# Activité du palier 2 : installer deux machines virtuelles avec VirtualBox

## Principe de l'activité

Cette activité se réalise à partir du dossier technique du palier 2.

Le dossier technique contient les informations nécessaires pour réussir le travail demandé.

L'activité ne redonne pas les explications techniques.
Vous devez donc consulter le dossier technique lorsque vous avez besoin d'une information.

## Travail demandé

Vous devez installer deux machines virtuelles avec VirtualBox :

1. une machine virtuelle Windows 11 Pro ;
2. une machine virtuelle Zorin OS.

Chaque machine virtuelle devra être :

- créée avec le bon nom ;
- configurée avec les bons paramètres ;
- installée avec la bonne image ISO ;
- accessible avec les identifiants communs demandés ;
- mise à jour ;
- équipée des additions invitées ;
- arrêtée proprement ;
- sauvegardée avec un instantané nommé correctement.

Le palier 2 est terminé uniquement lorsque les deux machines virtuelles sont prêtes pour le palier 3.

## Matériel et fichiers nécessaires

Avant de commencer, vérifiez que vous disposez des éléments nécessaires :

- un poste Debian 13 ;
- VirtualBox installé ;
- l'image ISO Windows 11 indiquée par le professeur ;
- l'image ISO Zorin OS indiquée par le professeur ;
- le dossier technique du palier 2.

Si une image ISO est absente, ne téléchargez rien au hasard.
Consultez le dossier technique, puis demandez au professeur si nécessaire.

## Demander de l'aide

Vous pouvez demander de l'aide, mais la demande doit être formulée correctement.

Avant d'appeler le professeur, vous devez pouvoir expliquer clairement :

- l'étape sur laquelle vous travaillez ;
- la machine virtuelle concernée ;
- ce que vous avez déjà essayé ;
- ce que vous avez observé ;
- ce qui ne fonctionne pas ;
- la partie du dossier technique que vous avez consultée.

Une demande d'aide ne doit pas être formulée comme ceci :

```text
Je ne comprends rien.
Ça ne marche pas.
Je ne sais pas quoi faire.
```

Une demande d'aide doit être formulée comme ceci :

```text
Je suis à l'étape de création de la machine virtuelle Windows 11 Pro.
J'ai sélectionné l'image ISO, mais je ne suis pas sûr du paramètre de mémoire vive à utiliser.
J'ai consulté le chapitre sur les paramètres complets de la machine virtuelle Windows 11 Pro.
J'ai besoin d'aide pour vérifier ma configuration avant de démarrer l'installation.
```

Le professeur aide plus efficacement lorsque le problème est expliqué avec précision.

Un technicien ne dit pas seulement que ça ne fonctionne pas.
Il explique ce qu'il a fait, ce qu'il a observé et ce qu'il veut vérifier.

## Étape 1 : préparer l'environnement

Ouvrez VirtualBox.

Vérifiez que vous pouvez créer une nouvelle machine virtuelle.

Repérez les images ISO nécessaires.

Ne commencez pas l'installation si vous n'avez pas les fichiers demandés.

## Étape 2 : créer la machine virtuelle Windows 11 Pro

Créez une nouvelle machine virtuelle pour Windows 11 Pro.

Utilisez le nom demandé dans le dossier technique.

Configurez la machine virtuelle avec les paramètres indiqués dans le dossier technique.

Associez l'image ISO Windows 11.

Avant de démarrer l'installation, vérifiez les points suivants :

- nom de la machine virtuelle ;
- type du système ;
- version du système ;
- image ISO sélectionnée ;
- mémoire vive ;
- nombre de processeurs ;
- disque virtuel ;
- stockage ;
- carte réseau ;
- paramètres liés à Windows 11.

Ne modifiez pas les paramètres au hasard.

## Étape 3 : installer Windows 11 Pro

Démarrez la machine virtuelle Windows 11 Pro.

Suivez l'installation jusqu'à l'arrivée sur le bureau Windows.

Pendant l'installation :

- choisissez l'édition demandée ;
- utilisez les identifiants communs indiqués dans le dossier technique ;
- ne créez pas de compte personnel ;
- utilisez un compte local selon la procédure indiquée dans le dossier technique.

Si Windows impose un compte Microsoft et que vous ne parvenez pas à créer le compte local, arrêtez-vous et appelez le professeur avec une demande formulée.

## Étape 4 : vérifier Windows 11 Pro

Une fois Windows installé :

- ouvrez la session avec le compte demandé ;
- vérifiez que le bureau s'affiche ;
- effectuez les mises à jour demandées ;
- redémarrez si nécessaire ;
- relancez la recherche de mises à jour si le dossier technique le demande ;
- arrêtez proprement Windows.

Ne fermez pas brutalement la fenêtre de la machine virtuelle.

## Étape 5 : installer les additions invitées sur Windows 11 Pro

Installez les additions invitées dans la machine virtuelle Windows 11 Pro.

Utilisez la procédure indiquée dans le dossier technique.

Après l'installation :

- redémarrez Windows ;
- vérifiez que l'affichage s'adapte mieux à la fenêtre ;
- arrêtez proprement Windows.

## Étape 6 : créer l'instantané Windows 11 Pro

Dans VirtualBox, créez un instantané de la machine virtuelle Windows 11 Pro.

Utilisez le nom demandé dans le dossier technique.

L'instantané doit être créé uniquement lorsque la machine virtuelle est dans un état propre.

## Étape 7 : créer la machine virtuelle Zorin OS

Créez une nouvelle machine virtuelle pour Zorin OS.

Utilisez le nom demandé dans le dossier technique.

- nombre de processeurs ;

## Étape 8 : installer Zorin OS

Pendant l'installation :

- utilisez les choix indiqués dans le dossier technique ;
- utilisez les identifiants communs demandés ;
- ne choisissez pas de nom personnel ;
- vérifiez que l'installation concerne uniquement le disque virtuel.

Si une option n'est pas claire, consultez le dossier technique avant de continuer.

## Étape 9 : vérifier Zorin OS

Une fois Zorin OS installé :

## Étape 11 : créer l'instantané Zorin OS

Dans VirtualBox, créez un instantané de la machine virtuelle Zorin OS.

Utilisez le nom demandé dans le dossier technique.

L'instantané doit être créé uniquement lorsque la machine virtuelle est dans un état propre.

## Étape 12 : vérifier le travail avant validation

Avant de demander la validation du palier, vérifiez les points suivants.

Pour la machine virtuelle Windows 11 Pro :

- la machine virtuelle existe ;
- elle démarre correctement ;
- le compte demandé fonctionne ;
- le mot de passe commun fonctionne ;
- le système est à jour ;
- les additions invitées sont installées ;
- la machine s'arrête proprement ;
- l'instantané demandé existe.

Pour la machine virtuelle Zorin OS :

- la machine virtuelle existe ;
- elle démarre correctement ;
- le compte demandé fonctionne ;
- le mot de passe commun fonctionne ;
- le système est à jour ;
- les additions invitées sont installées ;
- la machine s'arrête proprement ;
- l'instantané demandé existe.

## Résultat attendu

À la fin de l'activité :

- la machine virtuelle Windows 11 Pro est installée ;
- la machine virtuelle Zorin OS est installée ;
- les deux machines virtuelles utilisent les identifiants communs demandés ;
- les deux machines virtuelles sont à jour ;
- les additions invitées sont installées ;
- les deux machines virtuelles démarrent correctement ;
- les deux machines virtuelles s'arrêtent correctement ;
- chaque machine virtuelle possède un instantané propre ;
- les deux machines virtuelles sont prêtes pour le palier 3.

## Validation du palier

Le palier 2 est validé lorsque les deux machines virtuelles sont installées, accessibles, à jour, équipées des additions invitées et sauvegardées par un instantané.

Si une machine virtuelle ne démarre pas, si la session ne s'ouvre pas, si les additions invitées ne sont pas installées ou si l'instantané n'existe pas, le palier n'est pas terminé.

Vous devez corriger le problème, vérifier à nouveau, puis demander une nouvelle validation.
