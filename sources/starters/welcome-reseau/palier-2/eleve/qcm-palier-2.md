# QCM du palier 2 : installer deux machines virtuelles avec VirtualBox

## Consigne

Ce QCM vérifie que le dossier technique du palier 2 a été compris.

Pour chaque question, choisir une seule réponse : A, B ou C.

Vous devez enregistrer vos réponses dans un fichier texte nommé :

```text
qcm-palier2.txt
```

Le fichier doit contenir une réponse par ligne, sans espace entre le numéro de la question et la lettre choisie.

Exemple de format attendu :

```text
1a
2b
3c
4a
5b
```

Le fichier doit contenir les réponses aux 20 questions.

Le QCM doit être validé à 100 % avant de commencer l’activité.

## Validation

Le QCM est validé uniquement avec **20 bonnes réponses sur 20**.

Si une réponse est fausse :

1. relire le chapitre correspondant dans le dossier technique ;
2. corriger le fichier `qcm-palier2.txt` ;
3. demander une nouvelle validation.

Vous ne démarrez pas l’activité tant que le fichier `qcm-palier2.txt` n’est pas correct à 100 %.
